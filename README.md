#  PhishQuest

**PhishQuest** es un juego educativo interactivo diseñado para entrenar a los usuarios en la detección de correos electrónicos y sitios web fraudulentos (Phishing).

##  Características

*   **Entrenamiento Táctico:** Simulación de escenarios reales de ciberseguridad.
*   **5 Niveles de Dificultad:** Desde errores obvios hasta ataques sofisticados de ingeniería social.
*   **Feedback Inmediato:** Explicaciones detalladas de por qué un mensaje es seguro o sospechoso.
*   **Gamificación:** Sistema de puntuación, rachas de aciertos y ranking local.
*   **Interfaz Arcade:** Diseño visual inmersivo estilo retro-futurista.

##  Cómo Jugar

1.  Abre el archivo `index.html` en tu navegador web.
2.  Analiza el "Email Entrante" o la "URL" mostrada en pantalla.
3.  Decide si es **SEGURO** o **SOSPECHOSOS**.
4.  ¡Mantén tu racha para ganar bonificaciones y subir en el ranking!

##  Tecnologías

*   HTML5
*   CSS3 (Estilos personalizados, sin frameworks)
*   JavaScript (ES6+)
